## Run Locally

- Clone the project

```bash
  git clone https://github.com/Semaroam/capstone-backend.git
```

- Copy .ENV.EXAMPLE file to .env in root folder

- Install dependencies

```bash
  npm install
```

- Start the server

```bash
  npm run start-dev
```
- Go to local server
```bash
  http://localhost:8080/
```
